This research program proposes the development of AI-driven density functional theory methods for materials chemistry, combining adaptive machine learning approaches with rigorous quantum chemical foundations.

### Research Pillars

1. **Adaptive Density Functional Theory** -- Machine-learned system-specific corrections for unprecedented DFT accuracy
2. **ML Interatomic Potentials** -- Physics-informed MACE models for molecular property prediction
3. **AI-Driven Autonomous Chemistry** -- Agentic AI systems for accelerating computational materials discovery
